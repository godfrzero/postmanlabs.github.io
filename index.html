Open Source Projects from Postman Labs

Postman Interceptor

Helper extension for the Postman packaged app. Also helps send restricted headers. It allows you to use browser cookies, use all headers (even the ones restricted by XHR) like User-Agent, Content-Type etc and capture requests and send them to Postman!

https://github.com/postmanlabs/postman-chrome-interceptor

Newman

Newman is a command-line collection runner for Postman. It allows you to effortlessly run and test a Postman collection directly from the command-line. It is built with extensibility in mind so that you can easily integrate it with your continuous integration servers and build systems.

Newman maintains feature parity with Postman and allows you to run collections just the way they are executed inside the collection runner in Postman.

https://github.com/postmanlabs/newman

Newman Docker

Docker images for Newman Collection Runner.

https://github.com/postmanlabs/newman-docker

Sails MySQL Transactions

Sails ORM adapter for mySQL with transaction and replication support.

https://github.com/postmanlabs/sails-mysql-transactions

Async Domino

A module to allow time guaranteed asynchronous functions.

https://github.com/postmanlabs/async-domino

Env Lift

Simple namespaced environment variable configuration management solution.

https://github.com/postmanlabs/env-lift

Postman Schemas

Repository of all schemas for JSON structures compatible with Postman (such as the Postman Collection Format). The schemas are also hosted online, at schema.getpostman.com.

https://github.com/postmanlabs/schemas

Postman Chrome Extension Legacy Version

Postman REST Client Chrome Extension (Legacy Version) https://www.getpostman.com/.

This repository contains codebase for Postman Chrome Extension legacy version - v0.0.1 to v0.9.9. Updates pertaining to newer versions are not pushed to this repository.

https://github.com/postmanlabs/postman-chrome-extension-legacy

Postman Demo Server
